Overview

Flip'd is a 2D puzzle platformer built using C++ and the SDL multimedia library in the summer of 2012. The objective of the game is to collect coins by building platforms and walls. The twist is that you must build the map while upside down and then flip it to collect the coins. Coins are randomly placed on the map. After collecting all the coins in level, a new level with a larger area begins. As the map area increases it becomes increasingly difficult to design the map in such a way that all the coins can be collected in one go.

Controls

  • w, a, d keys to move the character.
  • Left click to build a block. Right click to remove it.
  • Press space to flip the map.
  • Press esc to pause the game.

Rules

  • Coins can only be collected when the map is not flipped.
  • The blocks can only be built when the map is flipped. In other words, you can not collect coins while building the map, or build the map while collecting coins.
  • Coins are reset each time the map is flipped. You must build the map in such a way that you can collect all the coins in one go.

State of Project

I built this game as a way to familiarize myself with SDL and the basics of game development. The game has a major issue where the jump distance is dependent on the computer the game is run on. I didn't notice this issue until tried the game on a Windows 8 laptop. Most Windows 7 computers should run the game fine. I haven't worked on this game since I finished it (summer 2012), and don't plan to revisit it.
